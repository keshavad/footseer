FootSeer is a tool designed for statistical football (soccer) prediction. It analyzes historical football leage data and various game metrics to determine likely outcomes through different predictive analytics techniques.

The user can enter specific game parameters, and FootSeer will predict the game outcome using each of the following analytical models:

Multinomial Logistic Regression- Calculates probability of win,lose or draw based on selected covariates. A LASSO was used to reduce the weights of each covariate so that the model is as simple as possible.

K-Nearest Neighbors- Groups together records into a class based on closeness of its predictors to the class predictors.

Decision Tree Model- Splits population of data into smaller and simpler branches to make better decisions.
